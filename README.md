# בלשי השפה

משחק תרגול שפה אינטרקטיבי לכיתה ד', בנוי מדפי העבודה של תגבור הקיץ (יולי 2026):
משפחות מילים ושורשים, סוגי מילים (פועל / שם עצם / שם תואר), המילה המתחזה,
מילות קישור ושמות תואר.

- ערכת נושא בלשית: כל נושא הוא "תיק חקירה", תשובה נכונה מקדמת את פענוח התיק.
- ריבוי פרופילים (שם + אווטאר, בלי סיסמה) — ההתקדמות נשמרת ב-Firestore ומשותפת בין מכשירים.
- גיימיפיקציה ידידותית לילדים: נקודות חקירה (XP), דרגות בלש, 1–3 כוכבים לכל תיק,
  רצף ימים עדין (בלי ענישה), תגים, לוח בלשים משפחתי.
- משוב מסביר על כל תשובה — גם כשטועים מקבלים את ההסבר, לא רק "טעות".

## הרצה מקומית

```bash
python -m venv .venv && .venv/bin/pip install -r requirements.txt
.venv/bin/uvicorn app.main:app --reload --port 8090   # backend
cd frontend && npm install && npm run dev              # frontend dev (proxy ל-8090)
```

בלי הרשאות Firestore, השרת עובר אוטומטית ל-store בזיכרון (לפיתוח בלבד).

## בדיקות

```bash
.venv/bin/python -m pytest -x
```
